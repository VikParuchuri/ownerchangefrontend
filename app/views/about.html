How does this work?

OwnerDraft uses machine learning and historical NFL data to calculate how a given combination of owner, GM, and coach would perform on a given team. Check out the code at Github for more details.

Why did you make this?

I've been a Washington fan for a while, and a disgruntled fan for approximately as long. I wanted to see if another owner would be better than Dan Snyder. Unfortunately, the data shows that almost every other owner is.

Can I help?

Absolutely! Check out the code for this site on Github, or the code for the machine learning part. Feel free to make a pull request if you want.
